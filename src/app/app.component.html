    Click on an item to remove it from the HashMap or enter a key and a value and and click Put to add it to the HashMap.

    HashMap Documentation

    hsMapData:HashMap<string, string> = new HashMap<string, string>(new AllFieldHashable<string>());
    this.hsMapData.add ("ABCDE", "Bird Seed");
    this.hsMapData.add ("FGHIJ", "Cat Food");
    this.hsMapData.add ("KLMNO", "Dog Food");
    this.hsMapData.remove ("ZZZZZ", "Not there anymore");
    for (let iter:JIterator<MapEntry<string, string> = this.hsMapData.entrySet().iterator(); iter.hasNext(); ) {
      let psp:MapEntry<string, string> = iter.next();
      console.log (psp.getKey() - psp.getValue());
    }

    How to iterate over a HashMap in the HTML template for an Angular Component:
    <li *ngFor="let psp of hsMapData.entrySet()"><button (click)="removeHashMap(psp);">{{psp.getKey() } } - {{psp.getValue() } } </button></li>
